# Projeto Canil - NodeJS + Typescript (b7web)

Projeto feito no módulo do curso NodeJS + Typescript da b7web - Bonieky Lacerda

### Pré-requisitos globais:
`npm i -g nodemon ts-node`

### Instalação
`npm install`

### Para rodar o projeto
`npm run start-dev`
